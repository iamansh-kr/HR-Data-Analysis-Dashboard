Project Overview
This project is a comprehensive HR Analytics Dashboard built in Microsoft Excel to analyze employee data and uncover actionable insights related to attrition, workforce demographics, and job satisfaction.
The dashboard transforms raw HR data into interactive visual insights to help organizations make data-driven decisions in areas like employee retention, hiring strategy, and workforce planning.
Objective
The primary goal of this project is to:
Analyze employee attrition patterns
Identify key factors affecting employee turnover
Provide HR teams with actionable insights
Build a visually interactive dashboard using Excel
